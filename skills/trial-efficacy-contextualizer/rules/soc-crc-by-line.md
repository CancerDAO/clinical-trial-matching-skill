# Standard of Care: Metastatic Colorectal Cancer (mCRC) by Line

Last updated: 2026-05. Use as efficacy benchmarks for vs-SoC comparison. Always verify against latest NCCN / CSCO guideline if patient is borderline between lines.

## 1L mCRC

Strategy depends on RAS status, BRAF V600E status, MSI status, sidedness (left vs right), and patient fitness.

### RAS wild-type, BRAF wild-type, MSS, left-sided

- **FOLFOX + cetuximab/panitumumab** — ORR 60-65%, mPFS 11-12 mo (CRYSTAL, FIRE-3, PARADIGM)
- **FOLFIRI + cetuximab/panitumumab** — similar efficacy, alternate backbone

### RAS mutant OR right-sided

- **FOLFOX + bevacizumab** — ORR 50-55%, mPFS 9-10 mo
- **FOLFIRI + bevacizumab** — alternate backbone
- **FOLFOXIRI + bevacizumab** (TRIBE) — ORR ~65%, mPFS 12 mo, mOS ~30 mo; for fit patients

### MSI-H / dMMR (any RAS status)

- **Pembrolizumab monotherapy** — KEYNOTE-177, mPFS 16.5 mo, ORR 43% (vs chemo mPFS 8 mo)
- **Nivolumab + ipilimumab** (CheckMate-8HW) — emerging as 1L preferred

### BRAF V600E (any RAS status)

- **FOLFOXIRI + bevacizumab** — induction; transition to BEACON regimen at progression
- **Encorafenib + cetuximab + binimetinib** in 1L is being studied; not yet standard

## 2L mCRC (post-1L progression)

### RAS wild-type, BRAF wild-type, post-FOLFOX/cetuximab

- **FOLFIRI ± bevacizumab** — ORR 4-15%, mPFS 4-7 mo
- **Aflibercept + FOLFIRI** (VELOUR) — ORR 19%, mPFS 6.9 mo
- **Ramucirumab + FOLFIRI** (RAISE) — ORR 13%, mPFS 5.7 mo

### RAS mutant, post-1L

- **FOLFIRI + bevacizumab** — switch chemotherapy backbone, continue anti-VEGF
- **CAPOX/XELOX + bevacizumab** — alternate

### MSI-H, post-1L (didn't get pembro 1L)

- **Pembrolizumab** — ORR ~30%, mPFS variable

### BRAF V600E, post-1L (BEACON regimen — second-line standard)

- **Encorafenib + cetuximab** ± **binimetinib** — ORR 20-26%, mPFS 4.3-4.5 mo, mOS 9-9.3 mo

### KRAS G12C, post-1L (FDA approved Jun 2024)

- **Sotorasib + panitumumab** (CodeBreaK 300) — ORR 26%, mPFS 5.6 mo, mOS 11.9 mo (vs investigator choice ORR 0%, mPFS 2.2 mo, mOS 10.3 mo)
- **Adagrasib + cetuximab** (KRYSTAL-10) — ORR ~46% in KRAS G12C 2L+ mCRC

## 3L mCRC

### Standard options (any subtype)

- **Regorafenib monotherapy** (CORRECT) — ORR 1-2%, mPFS 1.9 mo, mOS 6.4 mo (vs placebo 5.0 mo)
- **TAS-102 (trifluridine/tipiracil) ± bevacizumab** — RECOURSE: TAS-102 mono mOS 7.1 mo. SUNLIGHT: TAS-102 + bev mOS 10.8 mo, mPFS 5.6 mo, ORR 6%
- **Fruquintinib monotherapy** (FRESCO-2) — ORR 2%, mPFS 3.7 mo, mOS 7.4 mo (vs placebo 4.8 mo)

### KRAS G12C 3L (if not used 2L)

- **Sotorasib + panitumumab** — same as 2L; many CRC patients are 3L+ at the point KRAS G12C is identified
- **Adagrasib + cetuximab** — same

### MSI-H / dMMR 3L

- **Pembrolizumab** if not used earlier
- **Nivolumab ± ipilimumab** alternative

## 4L+ mCRC

Largely the unused options from 3L list. Decision becomes a feasibility/QoL question more than an efficacy question:

- **Regorafenib** if not used
- **TAS-102 ± bev** if not used
- **Fruquintinib** if not used
- **Re-challenge** with anti-EGFR (CHRONOS, CITRIC) for selected RAS/BRAF wild-type patients with detectable RAS clones cleared on liquid biopsy
- **Best supportive care** discussion if cumulative toxicity / declining KPS / median OS at this line typically <6 mo

## Patient-specific SoC selection

For PT-17CE02BC33 (KRAS G12C mCRC, post-FOLFOX/KELOX+camrelizumab+apatinib, currently L3 KELOX+camrelizumab+bevacizumab):

The patient's effective L4 SoC options are:

1. **Sotorasib + panitumumab** (CodeBreaK 300, FDA approved) — ORR 26%, mPFS 5.6 mo. **Best evidence base, mutation-targeted, off-trial accessible. Patient has not had anti-EGFR.**
2. **Regorafenib** — pivotal CORRECT, mPFS 1.9 mo. Less effective but well-tolerated for fit patients.
3. **TAS-102 + bevacizumab** — SUNLIGHT, mPFS 5.6 mo. Patient already on bevacizumab so familiar.
4. **Fruquintinib** — FRESCO-2, mPFS 3.7 mo. Chinese-developed, well-reimbursed in China.

**Trial pathways should be benchmarked against #1 (sotorasib+panitumumab)** as the highest-bar SoC for this patient's molecular subtype, NOT against #2/3/4 generic chemo-refractory benchmarks.

## Don't conflate

- **3L vs 4L data**: many "3L" benchmarks (CORRECT, FRESCO-2, RECOURSE) actually enrolled 3L+ patients; published mOS includes some 4L+ — be conservative when extrapolating
- **MSS vs MSI-H**: never apply MSI-H pembrolizumab data to MSS patients (PT-17CE02BC33 is MSS)
- **Chinese vs western data**: most Chinese CRC patients in trials have similar genetic profiles, but access to anti-EGFR may differ — fruquintinib particularly relevant in China
